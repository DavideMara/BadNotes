## QUESTIONS PROBLEMAS 1

**1.1** The **Job** of this layer is to manage **data exchange** within the local network by controlling the **sending** of **Frames**, and with the **address** it also **identifies** the devices **within** the **same network**.

**1.2** The transport layer provides end-to-end exchange between **processes** using ports.
* In specific it uses **2** types of connection: **TCP** that is connection-oriented and it has a controlled error-free flow.
* And then there is the **UDP**, that is connection-less and **unreliable** [NO ORDER/CONTROL].

**1.3** **Protocol** is a set of formal rules and conventions **governing** how two or more entities should exchange information **inside** a telecommunication system.

>**[Definition] PDU** (Protocol Data Unit) is the fundamental unit of data that is transferred between entities (or layers) **within** a computer network.

**1.6** The **OSI** model is a **7**-layer **architecture**, while the **TCP/IP** model **is a 4** layer model used for the **Internet**.

**1.10** A **router** is a networking device that **connects** computer networks and directs data traffic between them.

**1.11** **Fragmentation** is a process in computer networking where a **large** data packet, typically an IP **datagram**, **is** divided into smaller pieces called **fragments**. It is necessary **because** every network link has a **MTU** (Largest physical block of data it can handle).
