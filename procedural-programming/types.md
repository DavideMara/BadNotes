# Types

In the C language object refers to a location in the memory whose contents can represent values.

Objects that have names are also called variables.

## Object type

We can understand the type of object by 3 principal characteristics

| How much space occupies in memory | Values that a variable can have | The operation that can be performed on that variable |
| --- | --- | --- |

The `basic types` and the `enumerated types` **together** make up the `arithmetic types`

The `arithmetic types` and the `pointer types` **together** are called `Scalar types`

The `array types` and the `pointer type` are referred as the `aggregate types`
